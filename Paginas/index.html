Migueleiros é uma poderosa plataforma de estatísticas que permite aos usuários realizarem facilmente análises estatísticas, fornecendo uma saída interpretada com precisão e fácil de entender.

Plataforma de Estatística para leigos em estatísticas

Migueleiros é uma plataforma desenvolvida para facilitar o uso dos métodos estatísticos para pessoas que tem baixo conhecimento em cálculos estátisticos.

Saída de dados interpretada

O Migueleiros interpreta os dados estátisticos e gera gráficos e tabelas com precisão apartir dos dados inseridos.

Interface gráfica

O Migueleiros tem um desgin elegante e simples conduzindo os usuários passo a passo para uma fácil utilização do software.
